vim文本编辑器

Posted on 2020-03-29

Vim

三种模式 命令模式【 a / i / o】– 插入模式. a在光标前插入 i在光标后插入o换行插入

​ – 【esc】退出回到命令模式 【:】 – 底行模式

​ – 【esc】退出回到命令模式

底行模式：【:wq】 保存退出 【:w】先保存才能退出【 :q】 退出 【:wq!】强制保存退出 【:q!】不保存退出

vim index.html vim install.log 打开程序文件和日志文件

光标移动: hjkl左下上右 gg光标移动到第一行 G光标移动到最后一行

设置行号：:set nu 取消行号：set nonu

复制 yy 光标所在行 复制多行 nyy 光标向下n行

粘贴 p 粘贴到光标下一行

删除(剪切)字符 x. 删除多个字符nx

删除(剪切)行 dd 删除多行 ndd 包括光标所在行向下数n行

撤销 u 反撤销 ctrl + r

查找 / 例如 /root 查找root字符 再按 n 向下查找 N向上查找

替换

​ 全文替换 :%s/root/aaaaaa/g s开始 g结束 aaaaaa替换root

​ 范围替换：680,700s/root/aaaaa/g 680到700行执行替换

建立配置文件

vim ~/.vimrc

set nu设置行号

linusx命令

帮助命令 man

​ man ls 查看ls帮助手册

查找命令

​ whereis ls 查看ls命令的绝对路径的位置

​ find 查找文件所在的位置

​ find /root -name a.html 在root位置下查找是否有a.html

​ find /root -iname A.html 在root位置下查找是否有A.html

​ i 不区分大小写

​ grep “root” install.log 查找install.log文件内包括root字符串的行的内容

​ grep -i “root” install.log 同上不区分root大小写

​ grep -v “root” install.log 反向查找 查找不含有root的行

压缩与解压缩命令

.rar .zip windows常见压缩文件

.tar.gz .tar.bz2 linux常见压缩文件

压缩

.tar.gz

tar -zcvf php.tar.gz install.log install.log.syslog

z 标识gz格式的压缩文件

c 压缩

v 显示压缩过程

f 自定义

解压缩 tar -zxvf php.tar.gz

rm -rf install.log 删除install文件

定向解压缩 tar -zxvf php.tar.gz -C bbs -C后面跟路径解压到那个位置

.tar.bz2

tar -jcvf bbs.tar.bz2 bbs 压缩bbs这个目录

tar -jxvf bbs.tar.bz2 解压缩

tar -jxvf bbs.tar.bz2 -C bbs 定向解压缩

关闭和重启

shutdown -h now 关闭系统 管理服务器非常少，服务器基本不关机

reboot 重启系统 经常用到 更改配置文件后
